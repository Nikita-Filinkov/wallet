# Wallet API

Микросервис для управления электронными кошельками с поддержкой финансовых операций.

## Возможности

### Аутентификация и авторизация

- JWT-токены с хранением в HTTP-only cookies
- Регистрация новых пользователей
- Авторизация по email и паролю
- Механизм выхода из системы

### Управление кошельками

- Автоматическое создание кошелька при регистрации
- Просмотр баланса кошелька
- Пополнение баланса (deposit)
- Списание средств (withdraw)

### Безопасность и надёжность

- Изоляция транзакций уровня SERIALIZABLE
- Валидация всех входных данных
- Версионирование endpoints
---

## Технологический стек

- **Python 3.12**
- **FastAPI** — backend-фреймворк
- **SQLAlchemy 2.0** — асинхронная ORM
- **PostgreSQL 16** — СУБД
- **Alembic** — миграции
- **pytest, pytest-asyncio** — тестирование
- **Docker, Docker Compose** — контейнеризация и оркестрация
- **Poetry** — управление зависимостями
